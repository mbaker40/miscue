## `src/hud.ts` changes

Import `NodeOption, NodeKind, RouteSlot` from `./route` (types only).

### 1. `nodeChoice` — new signature (replaces the old one)
```ts
nodeChoice(depth: number, options: NodeOption[], fan: RouteSlot[], pick: (k: NodeKind) => void): void
```
- Buttons: one per option, same markup as today (`<b>title</b><small>desc</small>`, option `cls` appended to
  the button class). Clicking hides the panel and calls `pick(option.kind)`.
- **Forward fan**: below the buttons, when `fan.length > 0`, render a muted preview strip using the existing
  `tiny` class (no new CSS):
  ```html
  <div class="tiny">coming up · T5: Rack / Rest · T6: Rack / Re-chalk / ??? · T7: Rack / Re-chalk</div>
  ```
  i.e. for each fan slot: `T${slot.depth}: ` + option titles joined by ` / `; slots joined by ` · `.
- Heading: keep `<h2>Table ${depth}</h2>`; subtitle `Choose your table.`

### 2. `forge(options, onPick)` — new
```ts
forge(options: { label: string; desc: string }[], onPick: (i: number | null) => void): void
```
- Heading `The Forge` (h2), sub: `Press your soul against the wheel. One crack, one gift.`
- One `.btn pink` per option: `<b>${label} · +1 crack</b><small>${desc}</small>` → hide panel, `onPick(i)`.
- **Empty options = too fragile** (game refuses the killing crack): sub becomes
  `The wheel turns you away. One more crack would be your last.` and NO forge buttons.
- Always a final `.btn center` — `Walk away` → hide panel, `onPick(null)`.

### 3. `mystery(ev, onDone)` — new
```ts
mystery(ev: { title: string; desc: string }, onDone: () => void): void
```
- h2 `${ev.title}`, sub `${ev.desc}`, single `.btn center gold` `Move on` → hide panel, `onDone()`.

### 4. `wager(chalk, canCrack, actions)` — new
```ts
wager(chalk: number, canCrack: boolean, actions: { ante: () => void; anteCrack: () => void; decline: () => void }): void
```
- h2 `Money table`, sub: `Double chalk and a wider re-forge — if you can cover the ante.<br>You hold <b style="color:var(--gold)">◆ ${chalk}</b>.`
- `.btn gold`: `<b>Ante ◆ 5</b><small>Buy in. Elite-heavy rack, double payout.</small>` — rendered
  `disabled` when `chalk < 5`; click → hide panel, `actions.ante()`.
- Crack ante `.btn pink`, shown ONLY when `chalk < 5`:
  `<b>Ante a crack</b><small>Broke players pay in bone. +1 crack, buy in anyway.</small>` — rendered
  `disabled` unless `canCrack`; click → hide panel, `actions.anteCrack()`.
- `.btn center`: `<b>Walk on</b><small>Take a plain rack instead.</small>` → hide panel, `actions.decline()`.
- Note `wire()` only attaches to non-disabled buttons that exist — disabled buttons must simply not be wired
  (a disabled button with a `{once:true}` listener that can never fire is fine too, but don't rely on
  `disabled` alone: guard with the same conditions when wiring).

### Keep everything else as-is
`menu`, `draft`, `shop`, `rest`, `gameOver`, `victory`, `banner`, meters — unchanged. Match the file's
existing template-literal style.
